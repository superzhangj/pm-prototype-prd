# 🎨 PM Prototype PRD — 高保真交互原型 + 需求规格说明书生成器

## ✨ 核心功能

### 1. 高保真交互式 HTML 原型
- **真正可交互** — Tab 切换、弹窗开关、分页跳转、表单输入、下拉选择，全部真实可用
- **框选迭代工具** — 拖拽框选页面任意区域 → 填写修改描述 → 回传 AI 执行修改（无需重写整个页面）
- **工具栏** — 支持折叠/展开、拖拽到任意位置
- **多端自适应** — PC Web / APP / 小程序 / H5 全端覆盖

### 2. 自动生成原型注释
- **AI 自动分析每个功能区域** — 输入需求后，AI 自动识别页面上所有功能点（导航、搜索框、表格、按钮、弹窗、分页等），为每个功能区域生成对应的注释
- **五种注释类型智能分类** — 自动判断每条注释属于交互说明、业务逻辑、边界异常还是权限规则，使用不同颜色区分
- **墨刀式标注体验** — 每个功能区域自动生成彩色编号徽章，点击弹出详细说明卡片
- **右侧注释面板** — 按类型分组展示所有注释，点击自动导航到对应组件并高亮显示
- **丰富的交互操作** — 标记可拖拽重新定位、卡片可展开/折叠、描述可双击编辑、支持手动删除
- **新增注释** — 工具栏提供"添加注释"模式，点击页面任意位置即可创建新注释

### 3. 参考图 1:1 复刻
- 上传系统截图 → AI 自动提取色彩体系（主色/辅助色/背景/文字/边框）→ 输出 CSS 变量表 → 生成对齐原系统样式的原型
- 注释中自动标注哪些样式来自参考图复刻

### 4. 多端自适应
| 端类型 | 布局 | 交互特征 |
|-------|------|---------|
| PC Web | 1440px 容器，侧边栏+内容区 | hover、点击、分页 |
| APP | 375-414px 单列卡片 | 触摸手势、底部弹出 |
| 小程序 | 375-414px，微信原生导航 | tap 事件 |
| H5 | 响应式，安全区适配 | 触屏 tap 事件 |

### 5. 一键输出需求规格说明书 (PRD)
基于原型中的注释，自动输出完整的需求文档：

| 注释类型 | 自动映射到 PRD 章节 |
|---------|------------------|
| 🔵 交互说明 → | 功能需求详述 + 操作流程 + 交互规格表 |
| 🟠 业务逻辑 → | 业务流程 + 状态流转 + 数据逻辑 |
| 🔴 边界异常 → | 边界与异常处理（含处理策略） |
| 🟢 权限规则 → | 用户角色与权限矩阵 |
| 🟣 备注 → | 附录 / 数据字典 |

PRD 模板综合 **腾讯、阿里巴巴、京东** 等大厂格式，覆盖：
- 文档控制（版本记录、评审记录、术语表）
- 项目概述（背景、目标、范围、架构图）
- 角色权限（角色定义 + 权限矩阵）
- 功能需求详述（FR 条目，含验收标准）
- 数据字典（实体字段、枚举值、状态机）
- 非功能需求（性能/安全/兼容性/可用性）
- 验收标准 + 附录

## 🚀 快速开始

### 安装

1. 确保你已安装 [WorkBuddy](https://www.codebuddy.cn/)
2. 将本项目克隆或复制到 WorkBuddy Skills 目录：

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/pm-prototype-prd.git

# 复制到 WorkBuddy Skills 目录
# Windows:
copy -r pm-prototype-prd %USERPROFILE%\.workbuddy\skills\
# macOS/Linux:
cp -r pm-prototype-prd ~/.workbuddy/skills/
```

### 使用方式

在 WorkBuddy 对话中直接输入触发词，Skill 自动激活：

| 场景 | 触发词示例 |
|------|-----------|
| 画原型 | "帮我画个登录页面原型"、"设计一个后台管理系统" |
| 复刻系统 | "参考这个截图，仿照样式做个页面" |
| 加注释 | "给这个原型加注释说明" |
| 迭代修改 | "框选工具栏换个图标"、"把这个表格增加筛选条件" |
| 输出 PRD | "基于这个原型输出需求文档"、"生成PRD" |

## ⚙️ 技术特性

- **纯前端，零依赖** — 生成的 HTML 原型是单文件，无需服务器，双击即可运行
- **CSS 变量设计系统** — 所有样式使用 `:root` CSS 变量注入，更换主题只需修改变量值
- **`__addAnnotationOn()` 智能定位** — 基于 `getBoundingClientRect()` 自动计算注释标记位置，居中布局完美兼容
- **注释交互丰富** — 标记可拖拽、卡片可展开/折叠、描述可双击编辑、支持手动删除
- **框选回传机制** — 前端框选 → 格式化请求 → 复制到剪贴板 → 粘贴回传 AI → 自动执行修改
